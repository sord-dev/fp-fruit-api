# fp-fruit-repo

Ask for fruits and ye shall recieve.

## Main url - https://fruit-api.onrender.com/

- Get all fruits: GET /fruits/
- Show fruit: GET /fruits/{name}
- Delete fruit: DELETE /fruits/{id}
- Create fruit: POST /fruits/
